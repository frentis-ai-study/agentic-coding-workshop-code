# MCP 리소스 샘플 문서

이 문서는 MCP 파일 리소스 기능을 시연하기 위한 샘플 마크다운 파일입니다.

## 파일 리소스란?

파일 리소스는 실제 파일 시스템의 파일을 MCP를 통해 제공하는 방법입니다.

### 주요 특징

- **실시간 접근**: 파일이 변경되면 즉시 반영됩니다
- **보안 제한**: 지정된 디렉토리 외부 접근 차단
- **다양한 형식**: 텍스트, JSON, 마크다운 등 지원

### 사용 예시

```python
# 파일 리소스 등록
@mcp.resource("file://{filename}")
def get_file(filename: str) -> str:
    return read_file(filename)
```

## 보안 고려사항

파일 리소스를 제공할 때는 다음 보안 사항을 반드시 확인해야 합니다:

1. **경로 검증**: 절대 경로 차단
2. **디렉토리 제한**: 상위 디렉토리 탐색(`..`) 차단
3. **허용 범위**: 특정 폴더로 접근 제한
